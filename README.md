# 🚀 ProjectHub

ProjectHub is a modern project management and team collaboration platform inspired by Jira, Linear, ClickUp, and Trello.

It helps organizations, startups, freelancers, and students manage projects, track tasks, plan sprints, and collaborate efficiently from a single workspace.

## ✨ Features

* 📋 Project Management
* 📌 Scrum & Kanban Boards
* 🏃 Sprint Planning
* 👥 Team Collaboration
* 🔔 Real-time Notifications
* 📊 Analytics & Reporting
* 🏢 Organization Workspaces
* 🚀 Startup Workspaces
* 🎓 Student Workspaces
* 👨‍💻 Freelancer Workspaces
* 🌙 Dark & Light Theme

## 🛠️ Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS
* Shadcn UI
* Framer Motion

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL (Neon)

### ORM

* Prisma

### Authentication

* JWT

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/akhiljoseph10/Project-Hub.git
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Run Frontend

```bash
npm run dev
```

### Run Backend

```bash
npm run dev
```
